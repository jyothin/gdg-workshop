# Readme

### Env Information
- API Key (required)

## Create
```shell
$> pip install google-adk
$> adk create gdghyderabad-workshop
```

## Run
```shell
$> adk run gdghyderabad-workshop
```
OR
```shell
$> adk web --port 8000
```
